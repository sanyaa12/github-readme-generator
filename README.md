# GitHub README Generator

AI-powered GitHub README Generator using Streamlit and Groq API.

## Features
- Generate professional README files
- Uses GitHub repository data
- AI-generated markdown
- Download README instantly

## Tech Stack
- Python
- Streamlit
- Groq API
- GitHub API
- LLM
